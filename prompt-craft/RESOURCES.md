## Quick Reference

### Building a Hero

1. Get inspiration from **Supahero** or **Mobbin**
2. Screenshot what you like
3. Use **Iconify Solar** icons
4. Check colors with **Realtime Colors**
5. Animate with **GSAP** or **Motion**

### Building a Dashboard

1. Check **Mobbin** for real examples
2. Use **shadcn/ui** components
3. Icons from **Solar Duotone**
4. Charts from **Recharts**
5. Grid layout with **Bento** inspiration

### Standing Out

1. **Bricolage Grotesque** instead of Inter
2. **Solar Duotone** instead of Lucide
3. **Custom palette** from Huemint
4. **Animated background** from Unicorn Studio
5. **Mesh gradients** for depth
