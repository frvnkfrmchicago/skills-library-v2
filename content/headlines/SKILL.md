# Headlines & Titles

Write headlines that stop the scroll.

---

## Context Questions

Before writing headlines, ask:

1. **What platform is this for?** — Blog, YouTube, Instagram, newsletter, landing page
2. **What's the content type?** — How-to, listicle, case study, news, opinion
3. **Who's the audience?** — Beginners, experts, B2B, B2C, Gen Z
4. **What's the goal?** — Clicks, shares, SEO ranking, conversions
5. **What's the brand tone?** — Professional, casual, bold, educational

---

## Dimensions

| Dimension | Spectrum |
|-----------|----------|
| Length | Short/punchy ←→ Long/descriptive |
| Specificity | Broad appeal ←→ Hyper-targeted |
| Emotion | Rational/informative ←→ Emotional/curiosity |
| Format | Statement ←→ Question |
| Novelty | Evergreen ←→ Trending/timely |

---

## Derivation Logic

| If Context Is... | Then Consider... |
|------------------|------------------|
| Blog + SEO focus | Include keyword, use numbers, optimize for search |
| YouTube + entertainment | Bracket modifiers, emotional hooks, thumbnail pairing |
| Newsletter + warm audience | Personal, curiosity-driven, value-first |
| Instagram carousel + Gen Z | Short hook, contrarian takes, save-worthy |
| Landing page + conversion | Benefit-focused, clear value prop, urgency |

---

## TL;DR

| Platform | Best Practice |
|----------|---------------|
| **Blog** | Numbers + Power words + Specificity |
| **Carousel** | Hook slide first, curiosity gaps |
| **YouTube** | Brackets, emotional triggers |
| **Newsletter** | Personalization, value promise |

## 1. Power Words

### By Emotion

| Emotion | Words |
|---------|-------|
| **Urgency** | Now, Today, Deadline, Limited, Fast, Instant |
| **Curiosity** | Secret, Hidden, Revealed, Truth, Nobody tells you |
| **Authority** | Proven, Research-backed, Expert, Data shows |
| **Value** | Free, Save, Bonus, Exclusive, Unlock |
| **Fear** | Mistake, Avoid, Warning, Don't, Never |
| **Aspiration** | Transform, Master, Ultimate, Complete |

### Numbers That Work

| Number | Why It Works |
|--------|--------------|
| **3** | Easy to remember |
| **5** | Digestible list |
| **7** | Perceived completeness |
| **10** | Round, comprehensive |
| **21, 37, 101** | Specific = credible |

## 2. Blog Headlines

### Formula Patterns

| Pattern | Example |
|---------|---------|
| **[Number] Ways to [Outcome]** | 7 Ways to Double Your Email Open Rates |
| **How to [Outcome] (Even If [Objection])** | How to Build Muscle (Even If You're Over 40) |
| **The [Adjective] Guide to [Topic]** | The Complete Guide to TikTok SEO |
| **[Number] [Topic] Mistakes (And How to Fix Them)** | 5 Landing Page Mistakes (And How to Fix Them) |
| **Why [Common Belief] Is Wrong** | Why "Post Daily" Is Wrong for Growth |
| **[Topic]: What [Authority] Gets Right** | Pricing: What Apple Gets Right |
| **I [Did X] for [Time]. Here's What Happened** | I Posted Daily for 90 Days. Here's What Happened |

### Headlines That Rank

```
✅ Specific > Vague
"How to Write Cold Emails That Get 40% Reply Rates"
vs.
"How to Write Better Cold Emails"

✅ Current > Timeless (for news)
"Instagram Algorithm 2026: Complete Breakdown"
vs.
"How the Instagram Algorithm Works"

✅ Outcome > Feature
"Double Your Productivity with This System"
vs.
"The Best Productivity System"
```

## 3. Instagram Carousels

### Slide Structure

| Slide | Purpose | Formula |
|-------|---------|---------|
| **1 (Hook)** | Stop scroll | Bold claim, question, or contrarian take |
| **2-8 (Value)** | Deliver | One idea per slide, scannable |
| **9 (Summary)** | Recap | TL;DR of key points |
| **10 (CTA)** | Action | Save, share, follow, comment |

### Hook Slide Formulas

| Type | Example |
|------|---------|
| **Contrarian** | "Stop doing [common advice]" |
| **Curiosity Gap** | "The [topic] mistake 90% of people make" |
| **Direct Benefit** | "How I [achieved outcome] in [timeframe]" |
| **List Promise** | "[Number] [things] that will [outcome]" |
| **Question** | "Why aren't you [doing X]?" |
| **Authority** | "[X years] of [topic] in 10 slides" |

### Design Best Practices

```
✅ Text large enough to read on mobile
✅ Consistent brand colors
✅ One key point per slide
✅ Progress indicator (optional)
✅ Save-worthy value (people save for later)
```

## 4. YouTube Titles

### Formula Patterns

| Pattern | Example |
|---------|---------|
| **[Topic] [Bracket Modifier]** | iPhone 16 Review [After 30 Days] |
| **I [Did X] [Consequence]** | I Quit Coffee for 30 Days (Shocking Results) |
| **Why [Thing] Is [Unexpected]** | Why $20 Headphones Sound Better Than $500 |
| **[Number] [Topic] [Qualifier]** | 10 Mac Apps I Can't Live Without |
| **[Topic] — [Subtext]** | Morning Routine — Why I Wake at 4AM |

### Bracket Modifiers (CTR Boosters)

| Bracket | Effect |
|---------|--------|
| **[FULL GUIDE]** | Comprehensive signal |
| **[2026]** | Freshness signal |
| **[Honest Review]** | Trust signal |
| **[Watch Before Buying]** | Urgency |
| **[Step by Step]** | Beginner-friendly |
| **[Surprising Results]** | Curiosity |

### Title + Thumbnail Pairing

```
Title says WHAT → Thumbnail shows EMOTION

Title: "I Tried the $1 vs $100 Pizza"
Thumbnail: Split image, shocked face, clear contrast

Rule: Don't repeat the same words in both.
```

## 5. Newsletter Subject Lines

### Formula Patterns

| Pattern | Example |
|---------|---------|
| **[Direct Value]** | 3 tools I use to save 5 hours/week |
| **[Curiosity]** | The email that got me a $50K client |
| **[Personalized]** | {First_Name}, you're leaving money on the table |
| **[Urgency]** | Doors close tomorrow |
| **[Pattern Interrupt]** | 🚨 This is different |
| **[Question]** | Ready to 2x your output? |

### What Works (2026 Data)

| Element | Impact |
|---------|--------|
| **Personalization** | +26% open rate |
| **Emoji (1 max)** | +15% open rate |
| **Numbers** | +17% open rate |
| **35-50 characters** | Optimal length |
| **Lowercase** | Feels personal |

## 6. A/B Testing Headlines

### What to Test

| Element | Variations |
|---------|------------|
| **Number** | 5 vs 7 vs 10 |
| **Power word** | "Ultimate" vs "Complete" |
| **Format** | Question vs statement |
| **Specificity** | General vs hyper-specific |
| **Length** | Short vs long |

### Quick Test Method

```
1. Write 5-10 headline variations
2. Post same content with different headlines (different days/platforms)
3. Track: CTR, saves, shares
4. Document winning patterns
5. Build swipe file of winners
```

## 7. Headline Swipe File

### High-Performing Templates

```markdown
## Listicles
- [X] [Topic] Tips That Actually Work
- [X] Things I Wish I Knew About [Topic]
- [X] [Topic] Hacks for [Audience]

## How-To
- How to [Outcome] Without [Obstacle]
- How I [Achieved X] in [Timeframe]
- The Fastest Way to [Outcome]

## Comparison
- [X] vs [Y]: Which Is Better for [Use Case]?
- I Tested [X] and [Y]. Here's the Winner.
- [X] or [Y]? The Definitive Answer

## Case Study
- How [Person/Company] [Achieved X]
- [X] [Outcome]: Behind the Scenes
- What [Authority] Taught Me About [Topic]

## Problem-Solution
- [Problem]? Here's the Fix
- Why [Problem] Keeps Happening (And What to Do)
- The [Problem] Nobody Talks About
```
