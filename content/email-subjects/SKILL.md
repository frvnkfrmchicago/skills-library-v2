# Email Subject Lines

Write subject lines that get opened.

---

## Context Questions

Before writing subject lines, ask:

1. **What type of email?** — Newsletter, promotional, transactional, cold outreach
2. **What's the audience relationship?** — Cold, warm, loyal subscribers
3. **What's the primary goal?** — Open rate, click-through, purchase, reply
4. **What's the brand voice?** — Formal, casual, playful, urgent
5. **What device do they read on?** — Mobile-first, desktop, mixed

---

## Dimensions

| Dimension | Spectrum |
|-----------|----------|
| Length | Short (30 chars) ←→ Long (60 chars) |
| Personalization | Generic ←→ Highly personalized |
| Urgency | Soft/evergreen ←→ Time-sensitive |
| Tone | Professional ←→ Casual/playful |
| Risk | Safe/proven ←→ Bold/experimental |

---

## Derivation Logic

| If Context Is... | Then Consider... |
|------------------|------------------|
| Newsletter + warm list | Curiosity hooks, value-first, personal tone |
| Promotional + e-commerce | Urgency, discounts, FOMO, clear benefit |
| Cold outreach + B2B | Short, relevant, avoid spam triggers |
| Transactional + service | Clear, actionable, no marketing fluff |
| Re-engagement + dormant | Pattern interrupt, question, value reminder |

| Element | Best Practice |
|---------|---------------|

## 1. Subject Line Formulas

### High-Performing Patterns

| Pattern | Example | Best For |
|---------|---------|----------|
| **Direct Benefit** | "Save 5 hours this week with X" | Value-first |
| **Curiosity Gap** | "The email that changed everything" | High open, risky |
| **Question** | "Ready to 2x your output?" | Engagement |
| **FOMO/Urgency** | "Doors close tomorrow" | Time-sensitive |
| **Personal** | "{first_name}, you're leaving money on the table" | Relationship |
| **List/Number** | "5 tools I use daily" | Actionable |
| **How-to** | "How I write 10x faster" | Educational |

### Formula Templates

```
[Number] + [Benefit] + [Timeframe]
→ "3 ways to save 5 hours this week"

[Question] + [Implied Benefit]
→ "Ready to stop losing leads?"

{first_name} + [Personal Hook]
→ "{first_name}, quick question about your goals"

[FOMO Element] + [Action]
→ "Last chance: 50% off ends tonight"

[Curiosity] + [Partial Reveal]
→ "The tool nobody talks about (but everyone uses)"
```

## 2. Personalization Tokens

### Common Tokens

| Token | Example | Impact |
|-------|---------|--------|
| `{first_name}` | "Hey Sarah" | +26% open rate |
| `{company}` | "For Acme Inc" | B2B relevance |
| `{location}` | "Events in Austin" | Local relevance |
| `{last_action}` | "Your cart is waiting" | Behavioral |
| `{product_name}` | "About your order" | Transactional |

### Best Practices

```
✅ DO:
- Use first name at start OR middle (not both)
- Fallback to generic if empty: "Hey {first_name|there}"
- Test with/without personalization

❌ AVOID:
- ALL CAPS personalization: "HEY SARAH"
- Over-personalization (feels creepy)
- Using tokens without data quality
```

## 3. Emoji Guidelines

### What Works

| Emoji | Use Case | Notes |
|-------|----------|-------|
| 🔥 | Urgency, trending | Overused but works |
| 🚀 | Launch, new feature | Tech/startup |
| 📧 | Newsletter context | Meta but clear |
| ✨ | New, special | Positive energy |
| 🎉 | Celebration, announcement | Events |
| ⏰ | Deadline reminder | Urgency |
| 💰 | Money/savings | E-commerce |

### Rules

```
✅ DO:
- 1 emoji maximum per subject
- Place at start OR end (not middle)
- Test with/without

❌ AVOID:
- Multiple emojis (looks spammy)
- Random/unrelated emojis
- Emoji-only subjects
- 🔴 Red circle (looks like notification)
```

## 4. Length Optimization

### Character Limits

| Device | Visible Chars | Recommendation |
|--------|---------------|----------------|
| iPhone | 35-40 | Keep key info first |
| Android | 40-45 | Slightly more room |
| Desktop | 60+ | Full subject visible |
| Gmail | 70 | But most read on mobile |

### Mobile-First Approach

```
Put the hook in first 35 characters.

BAD:  "Our team has been working hard to bring you this amazing new feature..."
GOOD: "New feature: AI writes your emails"
```

## 5. A/B Testing

### What to Test

| Element | Variation A | Variation B |
|---------|-------------|-------------|
| **Length** | Short (30 chars) | Long (50 chars) |
| **Personalization** | With {first_name} | Without |
| **Emoji** | With emoji | Without |
| **Format** | Question | Statement |
| **Urgency** | With deadline | Without |
| **Case** | Title Case | lowercase |

### Testing Protocol

```
1. Test ONE variable at a time
2. Minimum 1,000 recipients per variant
3. Wait 24-48 hours for full data
4. Statistical significance: 95%+
5. Document winners in swipe file
```

## 6. Spam Trigger Avoidance

### Words to Avoid

| High Risk | Medium Risk | Context Matters |
|-----------|-------------|-----------------|
| FREE!!! | Act now | Free (lowercase ok) |
| $$$ | Limited time | Discount |
| URGENT | Don't miss | Offer |
| Congratulations | Winner | Sale |
| Click here | Best price | Deal |

### Format Triggers

```
❌ AVOID:
- ALL CAPS
- Excessive punctuation (!!!, ???)
- Misleading Re: or Fwd:
- Symbols like $$$
- Too many special characters

✅ INSTEAD:
- Normal capitalization
- Single punctuation
- Honest subject lines
- Words not symbols
```

## 7. Preview Text

The text that appears after the subject line on mobile.

### Optimization

| Do | Don't |
|----|-------|
| Extend the subject | Repeat the subject |
| Add context | "View in browser" |
| Create curiosity | Leave default |
| Include CTA hint | Use all caps |

### Examples

```
Subject: Your weekly roundup
Preview: 3 articles, 2 tools, and a surprise...

Subject: {first_name}, quick question
Preview: Takes 30 seconds to answer.

Subject: 🔥 New drop just landed
Preview: Limited stock. See what's new →
```

## 8. By Email Type

### Newsletter

```
- Value-first: "5 tips from this week"
- Curiosity: "What 1,000 marketers taught me"
- Personal: "A story I've never shared"
```

### Promotional

```
- Urgency: "24 hours left: 40% off"
- Benefit: "Save $50 on your next order"
- FOMO: "Selling fast: only 10 left"
```

### Transactional

```
- Clear: "Your order #12345 has shipped"
- Helpful: "Track your package"
- Actionable: "Confirm your email address"
```

### Cold Outreach

```
- Relevant: "Re: {company}'s growth"
- Curious: "Quick question about [topic]"
- Value: "Idea for {company}"
```
