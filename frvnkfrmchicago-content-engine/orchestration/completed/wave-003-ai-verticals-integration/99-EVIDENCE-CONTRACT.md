# Wave 003: Evidence Contract

To complete a lane, the assigned agent must rewrite their lane brief file (e.g. `01-ai-verticals-curation.md`) with the following completion evidence:

1. **Explainer**: Plain English explanation of the changes made and why.
2. **TL;DR**: High-level summary of modifications.
3. **Tables**:
   - Files changed (path, purpose)
   - Verification commands or tests executed
   - Remaining gaps (if any)
4. **Handoff row**: Update status in the Master Log.
