# Evidence Contract - Wave-002
Wave: wave-002

## Explainer
Each lane agent must rewrite its own brief file to serve as completion proof. Chat messages or text updates are not accepted as evidence.

## Required Sections
1. **Explainer**: Plain-language context of what changed.
2. **TL;DR**: Scannable summary of results.
3. **Delivery Summary Table**: Outlining requested outcomes vs. actual results with proof references.
4. **Files Changed Table**: List of files created/modified.
5. **Commands Run Table**: List of validation terminal commands run and their results.
6. **Artifacts Table**: Paths to new files.
7. **Remaining Gaps**: Any unfinished items or next actions.
