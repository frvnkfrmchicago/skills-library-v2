# Multi-Agent Plans

**Coordinate work across multiple AI agents.**

---

## How It Works

1. **Create a plan** → Define what each agent builds
2. **Store in this directory** → `direct-paths/multi-agent/[plan-name].md`
3. **Hand off to agents** → Copy each agent's section as their prompt
4. **Track progress** → Update status as work completes
