| Section | Purpose |
|---------|---------|

## Quick Links

### Handoffs
- Current handoffs go in `handoffs/`
- Each handoff is a standalone prompt ready to copy-paste

### Reviews  
- Review prompts for external AI agents
- Brutal, specific, actionable feedback templates

### Explainers
- Templates for explaining code
- Architecture walkthroughs
- Decision documentation

### Documentation
- Auto-doc patterns
- README generation
- API documentation

---

## How to Use

1. **Need another agent to build something?**
   - Copy a handoff template from `handoffs/`
   - Customize with your context
   - Paste to the other agent

2. **Want feedback on something you built?**
   - Copy a review prompt from `reviews/`
   - Send to external agent (Claude, GPT-4, etc.)
   - Get brutal feedback

3. **Need to explain code?**
   - Use templates in `explainers/`
   - Generate documentation as you build
