## Key Paradigm: Skills as Reasoning Frameworks


Skills should have:
1. **Frontmatter** — name, description, last_updated, owner
2. **Context Questions** — Ask before applying
3. **Dimensions** — Spectrums to explore

5. **Code Patterns** — Production-ready examples
6. **Related Skills** — Cross-references
