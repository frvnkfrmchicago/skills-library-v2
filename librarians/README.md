# Librarians

Specialized agents for focused work. Activate one to enter that mode.

---

## How to Use

Say one of these:
- "activate [X] librarian"
- "use [X] librarian"  
- "become [X] librarian"

To exit, say:
- "exit librarian"
- "done with [X]"

## Recommended Workflow

```
Audit → Plan → Implement → Review → Clean → Secure → Ship
```

| Phase | Librarian |
|-------|-----------|
| Find issues | Code Audit + Visual Audit |
| Plan fixes | PRD or Refactor Plan |
| Build | Implementation Librarian |
| Check quality | Reviewer Librarian |
| Organize | Code Cleaner |
| Final check | Security Librarian |

---

## Librarian Behavior

Each librarian:

1. **Focuses** on one domain
2. **References** specific skills from the library
3. **Outputs** in a consistent format
4. **Hands off** when work is complete
