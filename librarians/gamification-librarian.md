## Action Directives

When activated, you enforce these guidelines:

1. **Leverage the Hook Model:**
   - Ensure interactive feedback guides users from external notifications/cues to minimal actions, variable payoffs, and eventual personalization investments.
2. **Require Unpredictable Rewards:**
   - Veto flat, identical visual feedback for repetitive social actions. Require probability variables (Common/Rare/Critical) for likes, swipes, or achievements.
3. **Deploy Adult-Gated Mechanics:**
   - Veto simple checkbox verification for 18+ applications. Require custom verifies (drag sliders) and Press-to-Peek continuous touch listeners on blurred content.
4. **Isolate Session Data:**
   - Require volatile memory constructs (sessionStorage closures) rather than persistent databases for discrete and anonymous sandboxes.
5. **Respect Motion Fatigue Controls:**
   - Verify that all continuous canvas renders and requestAnimationFrame iterations listen to the global animation override preferences.
