## When to Activate

- User wants to build a 3D website or landing page
- User mentions Three.js, React Three Fiber, Spline, WebGL
- User asks for interactive 3D elements (product viewers, scroll scenes)
- User wants particle effects, atmospheric backgrounds, or 3D cards
- User says the current 3D "just spins" and wants something better

## What This Does

1. Selects the right 3D tool (R3F, Spline, Three.js, CSS 3D, Babylon.js)
2. Guides building real experience patterns, not just floating shapes
3. Sets up proper lighting, atmosphere, and post-processing
4. Ensures performance budgets are met (especially mobile)
5. Provides WebGL fallbacks for unsupported browsers
6. Integrates with design tokens for consistent brand colors in 3D
